DOCUMENTAÇÃO DE CASOS DE USO - TABELA
Título do caso de uso Cadastrar Usuário
Código Identificador DCUS01
Sumário Este requisito que o usuário faça o cadastro de seus dados no sistema.
Ator primário Usuário
Ator secundário Sistema
Pré-condições Registrar o usuário
Fluxo Principal
Ator Sistema
FP01 - O usuário seleciona "Cadastrar Usuário"  
  FP02 - Exibir a página "Dados do Usuário"
FP03 - O usuário informa o nome  
  FP04 - Receber o nome - FE01
FP05 - O usuário informa a data de nascimento  
  FP06 - Receber a data de nascimento - FE02
Fluxo de Alternativo FA01 - Cancela o cadastro de usuário.

  a. O usuário seleciona a opção "Limpar Dados".

  b. O caso de uso retorna a tela inicial.

  c. o caso de uso termina.
Fluxo de Exceção FE01 - Receber o nome.

  a. se no nome não for informado, o sistema exibe a mensagem: "informe o nome".

  b. Retorna ao FP03.
  FE02 - Receber a data de nascimento.

  a. se a data de nascimento não for informada, o sistema exibe a mensagem: "informe a data de nascimento".

  b. Retorna ao FP05.
Pós-condição Não há
Regras de Negócio RN01 - O cadastro poderá ser feito apenas para usuários maiores de 18 anos.
